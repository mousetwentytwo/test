Help improve GODspeed

Participate in its User Statistics and help improve the quality, reliability and performance of this software. If you choose to participate:

GODspeed will collect:
  • Anonymous usage data
  • Language settings
  • System information (OS and .NET Framework version)
But GODspeed will NOT:
  • Collect your name, address or any other personal information
  • Collect anything which could be used as a proof of piracy
  • Give away the collected information to any third party
You can stop participating at any time. Just go to the Help menu, select User Statistics Participation, then select No, I would not like to participate.
